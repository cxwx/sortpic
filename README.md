## 📖 算法特点

### 时间复杂度
- O(n²) 算法：插入排序、选择排序、冒泡排序等
- O(n log n) 算法：快速排序、堆排序、希尔排序等
- O(n log n) 最优：平滑排序

### 稳定性
- 稳定排序：插入排序、鸡尾酒排序
- 不稳定排序：快速排序、堆排序、选择排序
